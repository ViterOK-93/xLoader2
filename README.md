# xLoader2

A utility for flashing HEX files into Arduino UNO, Nano, and MEGA2560 boards. Simply select the HEX file, COM port, and click Upload. For older Arduino Nano boards, select 57600 baud rate.

![image.png](https://i.postimg.cc/L85jpsNM/image.png)

## Building

The project can be built using [SharpDevelop](https://github.com/icsharpcode/SharpDevelop) - a feature-rich IDE for .NET Framework 4.0 (15 MB installer / ~55 MB after installation)

Use version 4.4! Version 5 contains a ton of bugs.
